# maimai damage bot
An information bot for a certain washing machine game
## Installation
Requires Python 3.10+
```py
# venv
python -m venv venv
source venv/bin/activate         # Linux
venv/Scripts/activate            # Windows
# Deps
pip install -r requirements.txt
# Run
python main.py
```
